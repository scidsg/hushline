# EU Phase 0 (Deferred): Authoritative Source Policy and Adapter Plan

## Summary

Define the policy and implementation plan for expanding strict public-record coverage to EU jurisdictions after U.S. 50-state completion.

## Scope

- Build country-level authoritative source map (official regulators/bar authorities only).
- Define allowed domains and source label taxonomy for each country.
- Determine per-country adapter strategy and blocker risks (captcha, authentication, unstable URLs, legal constraints).
- Propose phased delivery order for EU countries.

## Acceptance Criteria

- Documented EU source-policy map with explicit allowed domains.
- Defined strict provenance requirements matching U.S. standard (record-specific URL, no private rankings).
- Risk and feasibility notes per country.
- Concrete implementation backlog created from this phase.

## Out of Scope

- Shipping EU data rows in this ticket.
- Modifying production dataset outside U.S.

## Dependency

- U.S. 50-state adapter rollout completed.
